6. REPORTE DE LOS RESULTADOS

6.1 Generación del reporte

Para generar el reporte con los datos del paciente y resultados de los estudios, se utiliza el botón Generar Reporte. Se le preguntará al usuario el nombre del archivo del reporte y de forma automática, el programa tomará las gráficas generadas para presentarlas en el reporte, al igual que los datos del paciente. Si no hay datos del paciente, los espacios quedarán vacíos en el reporte.

El reporte consta de 4 partes: Datos del paciente, señales de oscilaciones, áreas de oscilaciones, y parámetros.

6.2 Edición del reporte

La aplicación cuenta con un editor del reporte, el cual permite modificar el título, los subtítulos, y los parráfos del reporte. Se accede al editor con el botón Editar Reporte (ver Figura 7).

El editor solo permite modificar el texto de cada elemento del reporte, conservando el formato y ubicación.
