4. DATOS DEL PACIENTE

Para iniciar, el usuario puede seleccionar entre introducir los datos de un paciente nuevo con el botón Nuevo Paciente, o cargar los datos a partir de un archivo de texto con un formato preestablecido, con el botón Cargar Datos. Con cualquiera de las 2 opciones, se habilitarán los campos de datos del paciente y el botón Guardar Datos (ver Figura 2).

Solo se podrán guardar los datos si todos los campos están completos. Si hace falta algún campo, se le informará al usuario.

Figura 2. Datos del paciente.

Los datos se guardan en un formato específico, en un archivo de texto. Este formato está ajustado para que pueda ser cargado por la aplicación con el botón Cargar Datos
