6. REPORT OF THE RESULTS

6.1 Report generation

To generate the report with the patient data and results of the studies, the button Generate Report is used. The user will be asked the name of the report file and automatically, the program will take the generated graphics to present in the report, as well as the patient's data. If there is no patient data, the spaces will be empty in the report.

The report consists of 4 parts: Patient data, oscillation signals, oscillation areas, and parameters.

6.2 Report edition

The application has a report editor, which allows modifying the title, the subtitles, and the paragraphs of the report. The editor is accessed with the button Edit Report (see Figure 7).

The editor only allows to modify the text of each element of the report, keeping the format and location.
