4. PATIENT'S DATA

To start, the user can choose between enter the data of a new patient with the button New Patient, or to load the data from a text file with a predetermined format, with the button Load Data. With any of the 2 options, the data fields and the button Save Data will be enabled (see Figure 2).

The data can only be saved if all the data fields are complete. If any data field is missing, the user will be informed.

Figure 2. Patient's data.

The data is saved in a specific format, in a text file. This format is set so that it can be loaded by the application with the button Load Data.
